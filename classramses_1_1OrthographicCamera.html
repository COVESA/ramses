Detailed Description

The OrthographicCamera is a local camera which defines an orthographic view into the scene.

A valid camera for rendering must have viewport and frustum set, see ramses::Camera for ways to set these parameters.

Examples
ramses-example-basic-renderpasses/src/main.cpp, ramses-example-dcsm-provider/src/main.cpp, ramses-example-local-client-dcsm/src/main.cpp, ramses-example-local-offscreenbuffer/src/main.cpp, ramses-example-local-pick-handling/src/main.cpp, ramses-example-text-basic/src/main.cpp, and ramses-example-text-languages/src/main.cpp.
